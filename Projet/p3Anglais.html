Project 3

Implementation of the k-NN Algorithm

This project focuses on the implementation and analysis of the k-nearest neighbors (k-NN) algorithm applied to datasets such as iris and Pok√©mon.

We evaluate different distance metrics (Manhattan, Euclidean), test various choices of k, and analyze the robustness of the model through cross-validation.

Finally, we assess the efficiency of the data structures used to store and process information.

Skills Learned

  • Implementation of k-NN: Development of a dedicated class managing distance computation, data normalization, and classification.
  • Cross-validation: Setting up cross-validation to assess the robustness of the model.
  • Choosing the best k: Comparing performance for different values of k and different distance metrics.
  • Algorithmic efficiency: Analysis of the data structures used and optimization of traversal methods to improve speed.
  • Error prevention: Implementation of techniques to avoid overfitting and ensure result reliability.
  • Performance evaluation: Analysis of obtained results and selection of the optimal model.
